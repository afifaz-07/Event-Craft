Event Management System Project Description:
This is an Event Management System developed in C language using arrays, functions, loops, switch cases, and conditional statements. The system allows both clients and admins to manage events efficiently, with features for signup/login, event creation, tracking, budget management, guest handling, and complaint resolution.

Features:
Admin Login,
Client Signup,
Client Login,
User Verification (Approve/Reject by Admin),
Add Event,
View Events,
Search Event,
Update Event,
Delete Event,
Track Events,
Budget Management,
Guest Information,
Complaint Management.
